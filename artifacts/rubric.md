# Five-Axis Evaluation Rubric — Climate Records Campaign

## Axis Definitions

| Axis | Pass Criterion | Fail Criterion |
|---|---|---|
| TRUE | Reproduced from published artifact (same number ±float tolerance) | Cannot reproduce; code missing; number changes |
| NOVEL | Not stated in Wikipedia/IPCC/textbook for this dataset | Exact claim found in public source |
| SURPRISING | Result is outside registered prior (prior written BEFORE examining data) | Result inside prior CI; or prior written after seeing result |
| HONEST | Uncertainty quantified; ≥1 weakening analysis included; no cherry-picking | No uncertainty; cherry-picked window/city/year |
| CLEAR | Hero artifact is self-explanatory (claim + evidence + uncertainty visible) | Requires reading node body to understand artifact |

## Baseline (this campaign)
- Global mean land warming: +1.08°C ± 0.16°C (1850–1880 vs 1981–2013, n=208 countries)
- Pre-1850 uncertainty floor: ≥1.5°C per month

## Prior Art (do NOT claim as surprising)
- Arctic amplification (high-latitude countries warm 2× faster)
- DTR decline (min warming faster than max, Karl et al. 1993)
- Tambora/Year Without a Summer 1816
- Global warming ~1°C since pre-industrial

## Scoring
- GREEN: 5/5 axes pass
- YELLOW: 4/5 (one marginal)  
- RED: ≤3/5 or any hard fail on TRUE/HONEST
